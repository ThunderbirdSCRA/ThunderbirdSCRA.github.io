# Thunderbird Chapter – Signal Corps Regimental Association (SCRA)

This repository contains the source code for the official website of the **Thunderbird Chapter of the Signal Corps Regimental Association (SCRA)**.

The site is a modern static web application built with **React, TypeScript, and Vite**, and is published using **GitHub Pages**.

## 🌐 Live Website

The production site is available at:

https://thunderscra.org

## 🎯 Purpose of the Site

The Thunderbird Chapter website exists to:

- Provide information about the chapter’s mission, leadership, and events
- Support engagement with active duty, reserve, guard, veteran, and civilian Signal Corps members
- Preserve and share the heritage and traditions of the U.S. Army Signal Corps
- Serve as a public-facing communication platform for the Thunderbird Chapter

## 🛠️ Technology Stack

The site is built using the following technologies:

- TypeScript
- React
- Vite
- HTML
- CSS
- GitHub Pages (hosting and deployment)

This is a fully static site with no server-side backend.

## ✅ Prerequisites (for Local Development)

To run this site locally, you will need **Node.js**, which includes **npm** (Node Package Manager).

Node.js is required **only** if you wish to run or build the site on your own computer.  
You do **not** need Node.js to view or use the published website.

Node.js can be downloaded from:

https://nodejs.org

Installing the **LTS (Long Term Support)** version is recommended.

## 🚀 Local Development

To run the site locally for development or testing:

1. Clone the repository
2. Install dependencies
3. Start the Vite development server

Run the following commands:

```
git clone https://github.com/ThunderbirdSCRA/ThunderbirdSCRA.github.io.git
cd ThunderbirdSCRA.github.io
npm install
npm run dev
```

Once running, the site will be available at the local development URL provided by Vite (typically http://localhost:5173).

## 🏗️ Build for Production

To create a production build of the site, run:

```
npm run build
```

The output will be generated in the `dist/` directory, which is used for deployment to GitHub Pages.

## 📦 Deployment & Hosting

This site is hosted and published using **GitHub Pages**.

- The repository is configured to build and publish the site automatically
- Builds generated by Vite are deployed via GitHub Pages
- The custom domain **thunderscra.org** is configured through the repository’s GitHub Pages settings

Any updates merged into the main branch will trigger a new site deployment.

## 🔁 Forking & Customization Notes

If you fork this repository and intend to use it for **your own website**, additional setup and customization will be required.

In particular:

- You will need to review and update **GitHub Pages settings** in your fork
- If you want to use your **own custom domain**, you must configure it in your repository’s Pages settings
- DNS records for your domain must be adjusted through your domain registrar
- You may need to update or remove any existing domain-specific configuration

These steps are not handled automatically when forking the repository.

For detailed guidance, refer to the official GitHub Pages documentation on managing custom domains:

https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site/managing-a-custom-domain-for-your-github-pages-site
